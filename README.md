# 🔍 User Explorer App

A responsive JavaScript application that fetches and filters users using a REST API.

## 🚀 Features
- Fetch users from API
- Search users by name
- Loading state
- Error handling
- Clean UI

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript (Fetch API)

## 📌 How to Use
1. Click the search button to load users  
2. Type a name in the input field  
3. Matching users will be displayed
